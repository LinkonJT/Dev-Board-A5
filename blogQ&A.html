Question-1: What are the different ways to select an element in the DOM?
we can an element in the DOM by calling it by its ID, classname, tagname. we can use .getElementById(), getElementsByClassName(), querySelectorAll(), querySelector() etc to call those elements.
Question-2: What is the difference between innerHTML, innerText, and textContent ?
the main difference beetween these three atributes are that, innerHTML works with HTML attributes like div, p , h1, h2, h3 etc. To manipulate these inside and element we can use element.innerHTML.
innerText is used to set or manipulate the text contents of an element, and it returns results considering the CSS style, JavaScript code.
But, textContent returns all texts, including hidden texts in an element, without considering the CSS style or JS codes.
Question-3: What is event delegation in the DOM?
In the DOM event delegation is a technique to add an event listener to the child elements by using the event handler of their parent element.
Question-4: What is event bubbling in the DOM?
Event bubble is a mechanism in the DOM where the event handler first works for the innermost element and then goes up to the top of the DOM tree. so, it first works with the lowest child element and then goes up to the parent element.
Question-5: How do you create, add, and remove elements using JavaScript?
we can use .createElement(tagName) to create div, p, span etc.
we can use .appendChild() to add elements to the parent element.
we can use .removeChild() to remove elements from the parent element.
